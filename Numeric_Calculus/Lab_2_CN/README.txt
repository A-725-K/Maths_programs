ISTRUZIONI PER LA COMPILAZIONE DEL LABORATORIO 2:
	- 1° esercizio: digitare da linea di comando
								g++ ex1.cpp -o ES1 -Wall
	- 2° esercizio: digitare da linea di comando
								g++ ex2cn.cpp funzioni_matrici.cpp funzioni_matrici.h -o ES2 -Wall
	- 3° esercizio: digitare da linea di comando
								g++ ex3cn.cpp funzioni_matrici.cpp funzioni_matrici.h -o ES3 -Wall

- Il file funzioni_matrici.cpp contiene l'implementazioni delle funzioni utilizzate nei nostri programmi.
- Il file funzioni_matrici.h contiene le definizioni delle funzioni e serve da "libreria".
- I file ex1.cpp, ex2.cpp e ex3.cpp contengono i main dei programmi.
- Il file CondizionamentoMatrici.m e' uno script di Matlab che calcola il condizionamento delle matrici prese in esame negli esercizi.
